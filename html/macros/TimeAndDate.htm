Macro Reference - Time And Date

日付け・時刻のマクロは下表の通りです。ほとんどは2桁のlong型です。

全てのマクロの表はこちら。

 

マクロ 説明
Macro Description
@MSEC クロックのミリ秒値。 00から999の範囲。
@SEC クロックの秒数： 00-59
@MIN クロックの分数 ：00-59
@HOUR 24時間表示でのクロックの時刻： 00-23
@MDAY 現在の日付： 01-31
@MON 現在の月： 01-12
@YEAR 現在の年（西暦で4桁）
@WDAY 曜日： 1から7までの数値が日曜から土曜日までと対応している。
@YDAY 1年における日付： 001-366 (うるう年でない場合、001-365)
