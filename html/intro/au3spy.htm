AutoIt Window Information Tool

AutoIt v3にはAutoIt Window Info Tool (Program Files\AutoIt3\AU3Info.exe)というツールが付属しています。  AU3Infoを使うことで特定のウィンドウの情報を得ることができ、効果的に自動化をおこなうことができます。  得られる情報は次の通りです。

  •  ウィンドウタイトル
  •  (表示・非表示されている)ウィンドウ上のテキスト
  •  ウィンドウのサイズと座標
  •  ステータスバーの内容
  •  マウスポインタ座標
  •  マウスポインタ位置のピクセルの色
  •  マウスポインタ位置のControlの詳細

AU3Infoを使用するには(コマンドラインもしくはスタートメニューから)ただ実行するだけです。  AU3Infoは表示内容を読めるよう常に画面最前面に表示されます。  情報を取得したいウィンドウを動かすかアクティブ化すると・・・AU3Infoの表示内容が利用可能な情報に更新されます。  AU3Infoの手助けによりすぐに自動化をおこなえます!

 

AU3Info使用中にテキストを直接コピーしたい場合はスペルミスをしないようにCTRL-Cを使ってAU3Infoのテキストをコピーし、スクリプトにペーストします。  リストビューとして情報を持っているタブでは項目をダブルクリックするだけでクリップボードにコピーできます。 もし取得したい情報がピクセル/マウス情報の場合、表示内容が更新され続けると情報取得は難しいです!  この問題を解決するためにCTRL-ALT-Fを押すことでAU3Infoの出力を"フリーズ"することができます。  "フリーズを解除"するにはもう一度同じキーを押してください。
