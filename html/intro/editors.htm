Script Editors

AutoItスクリプトはどのようなテキストエディタでも編集できるシンプルなテキストファイルです。しかしながらAutoItのキーワードと関数を目立つよう強調表示してスクリプトを書くことをより簡単に、またつづり間違いを少なくしてくれるようなエディタが存在します。その多くはフリーやシェアウェアです。

現在、多くのAutoItユーザーが使用しているのはSciteです。AutoIt開発チームはすべての語句を強調表示し、多くのサードパーティー製のAutoItツールを組み込んだカスタム版のSciteを作成しました。"ライト"版のSciteはAutoItのインストールパッケージに含まれています。全てのツールを含む完全AutoIt版のSciteはhttp://www.autoitscript.com/autoit3/scite/からダウンロードできます。

 

他のおすすめのエディタには以下のものがあります。

  • TextPad
  • Crimson Editor (フリー)
  • Source Edit (フリー)
  • UltraEdit

AutoItのExtraフォルダ下には多くのエディタ用設定ファイルがあります。 (Start Menu / AutoIt v3 / Extra下)。
