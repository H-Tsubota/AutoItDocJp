Function Reference

String

値の文字列表現を返します。

String ( expression )

 

パラメータ

expression 文字列に変換する値

 

返し値

文字列を返します。

 

注意

文字列の最大長は2147483647文字です(ただしAutoItスクリプトの1行の最大長は4095文字であることを憶えておいてください)。

 

関連

Int, Number, IsString, Chr, BinaryToString, ChrW, HWnd, Ptr, StringToBinary

 

例


$var = String(10)
;$varは文字列"10"になる
