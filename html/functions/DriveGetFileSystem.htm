Function Reference

DriveGetFileSystem

ドライブのファイルシステムタイプを返します。

DriveGetFileSystem ( "path" )

 

パラメータ

path 調べたいドライブのパス

 

返し値

成功 ドライブのファイルシステムタイプを文字列で返します。下表参照。
失敗 @errorを１に設定します。


返し値 意味
1 (数値) ドライブにメディア(CD, Floppy, Zip)が接続されていないか、メディアがフォーマットされていない(RAW)。
"FAT" フロッピー、RAMディスク、USB"pen"ドライブなど、500MB以下のドライブに多くみられるのファイルシステム。
"FAT32" Windows 9x/Meのハードディスクドライブにみられるファイルシステム。
"NTFS" Windows NT/2000/XPのハードディスクドライブにみられるファイルシステム。
"NWFS" Novell Netwareファイルサーバーにみられるファイルシステム。
"CDFS" CD(もしくはバーチャルCDドライブとしてマウントされているISOイメージ)を表す。
"UDF" DVDを表す。

 

注意

返し値のリストにない値が返される場合があります。

 

関連

DriveGetDrive, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

 

例


$var = DriveGetFileSystem( "c:\" )
MsgBox(4096,"File System Type:", $var)
