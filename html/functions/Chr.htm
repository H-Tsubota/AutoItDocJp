Function Reference

Chr

ASCIIコードに対応した文字を返します。

Chr ( ASCIIcode )

 

パラメータ

ASCIIcode 0-255の範囲のASCIIコード(例：65 は大文字のA を返します。)

 

返し値

成功 与えられたASCIIコードに対応する文字の文字列を返します。
失敗 ASCIIcodeが255より大きい場合、空文字列を返し、@errorを1に設定します。

 

注意

Chr(48) == "0", Chr(57) == "9", Chr(65) == "A", Chr(90) == "Z", Chr(97) == "a", Chr(122) = "z", etc.
完全なASCIIコード表は付録を参照。

 

関連

Asc, AscW, ChrW, String

 

例


$text = ""
For $i = 65 to 90
    $text = $text & Chr($i)
Next
MsgBox(0, "Uppercase alphabet", $text)
