Function Reference

WinExists

指定したウィンドウが存在するかどうかを調べます。

WinExists ( "title" [, "text"] )

 

パラメータ

title 調べるウィンドウの名前。ウィンドウ名の特別な記述の仕方参照
text [オプション] 調べるウィンドウのテキスト

 

返し値

成功 ウィンドウが存在する場合1を返します。
失敗 それ以外の場合0を返します。

 

注意

ウィンドウが存在する場合、たとえ隠されていても1を返します。

 

関連

WinActive, WinWait, WinWaitActive, WinWaitClose, WinWaitNotActive, WinTitleMatchMode (Option), ProcessExists, WinClose

 

例


If WinExists("[CLASS:Notepad]") Then
    MsgBox(0, "", "Window exists")
EndIf
