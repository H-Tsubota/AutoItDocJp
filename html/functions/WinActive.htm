Function Reference

WinActive

指定したウィンドウが存在するか、現在アクティブかを調べます。

WinActive ( "title" [, "text"] )

 

パラメータ

title 調べるウィンドウの名前。ウィンドウ名の特別な記述の仕方参照
text [オプション] 調べるウィンドウのテキスト

 

返し値

成功 ウィンドウがアクティブだった場合、ウィンドウのハンドルを返します。
失敗 それ以外の場合0を返します。

 

注意

なし。

 

関連

WinExists, WinWait, WinWaitActive, WinWaitClose, WinWaitNotActive, WinTitleMatchMode (Option)

 

例


If WinActive("[CLASS:Notepad]") Then
    MsgBox(0, "", "Window was active")
EndIf
