Function Reference

IsAdmin

現在のユーザーが完全な管理者権限を持っているかどうか調べます。

IsAdmin ( )

 

パラメータ

なし。

 

返し値

成功 現在のユーザーが管理者権限を持っている場合1を返します。
失敗 管理者権限を持っていない場合0を返します。

 

注意

Windows Vistaの場合、完全な管理者権限で実行されている場合(例えば#RequireAdminが使用されているか、既にUACによって承認されている場合)のみ1が返されます。

 

関連

#RequireAdmin

 

例


If IsAdmin() Then MsgBox(0, "", "Admin rights detected")
