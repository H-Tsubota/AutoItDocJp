Function Reference

StringLeft

文字列の左端から指定した文字数分を返します。

StringLeft ( "string", count )

 

パラメータ

string 対象となる文字列
count 取得する文字数

 

返し値

対象となる文字列の 左端 から指定した文字数文取得した文字列を返します。

 

注意

countパラメータが文字列の長さを超えた場合、文字列全体が返されます。
countパラメータが0より小さい場合、空の文字列が返されます。

 

関連

StringInStr, StringLen, StringLower, StringMid, StringRight, StringTrimLeft, StringTrimRight, StringUpper, StringCompare, StringReplace, StringSplit

 

例


$result = StringLeft("I am a string", 3)
MsgBox(0, "Leftmost 3 characters are:", $result)
