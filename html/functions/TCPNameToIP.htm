Function Reference

TCPNameToIP

インターネット名をIPアドレスに変換します。

TCPNameToIP ( name )

 

パラメータ

name インターネット名を格納した文字列

 

返し値

成功 nameに対応するIPアドレスを格納した文字列を返します。
失敗 ""を返し、@errorを設定します。
@error: WindowsAPIのWSAGetErrorの返し値 (MSDNを参照)。

 

注意

実行前にTCPStartupの実行が必要です。

 

関連

TCPStartup

 

例


TCPStartup()

msgbox(0,"IP???",TCPNameToIP("TheMachineNextToYOU"))
