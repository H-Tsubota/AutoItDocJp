Function Reference

DriveGetDrive

ドライブを列挙した配列を返します。

DriveGetDrive ( "type" )

 

パラメータ

type 検索するドライブのタイプ。
"ALL"、"CDROM"、"REMOVABLE"、"FIXED"、"NETWORK"、"RAMDISK"、"UNKNOWN"のいずれか。

 

返し値

成功 見つかったドライブ名(コロンの付いたドライブを表す文字)を格納した配列を返します。ゼロ番目の要素にはドライブ数が格納されています。
失敗 "" を返し、@errorを1に設定します。

 

注意

なし。

 

関連

DriveGetFileSystem, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

 

例


$var = DriveGetDrive( "all" )
If NOT @error Then
    MsgBox(4096,"", "Found " & $var[0] & " drives")
    For $i = 1 to $var[0]
        MsgBox(4096,"Drive " & $i, $var[$i])
    Next
EndIf
