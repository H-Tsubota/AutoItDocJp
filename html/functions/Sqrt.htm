Function Reference

Sqrt

平方根を計算します。

Sqrt ( expression )

 

パラメータ

expression 平方根を計算したい負でない値

 

返し値

成功 平方根が返されます。
失敗 パラメータが負の値の場合、-1.#INDを返します。

 

注意

n乗根を計算したい場合は累乗演算子を使用してください。 x ^ (1/n)
MsgBox(0,"The cube root of 27 is", 27 ^ (1/3) )

 

関連

なし。

 

例


$x = Sqrt(2)  ;1.4142135623731を返します。
$y = sqrt(9)  ;3を返します。
