Function Reference

StringCompare

2つの文字列をオプション付きで比較します。

StringCompare ( "string1", "string2" [, casesense] )

 

パラメータ

string1 1番目の評価する文字列
string2 2番目の評価する文字列
casesense [オプション] 処理時で大文字小文字を区別するかどうかを表すフラグ
0 = 区別しない。ユーザーのロケールを使用(デフォルト)
1 = 区別する。
2 = 区別しない。基本的/高速な比較を使用する。

 

返し値

0 string1とstring2は等しいです。
> 0 string1はstring2よりも大きいです。
< 0 string1はstring2よりも小さいです。

 

注意

なし。

 

関連

StringInStr, StringLeft, StringLen, StringLower, StringMid, StringRight, StringTrimLeft, StringTrimRight, StringUpper

 

例


$result = StringCompare("MELON", "melon")
MsgBox(0, "StringCompare Result (mode 0):", $result)

$result = StringCompare("MELON", "melon", 1)
MsgBox(0, "StringCompare Result (mode 1):", $result)

$result = StringCompare("MELON", "melon", 2)
MsgBox(0, "StringCompare Result (mode 2):", $result)
