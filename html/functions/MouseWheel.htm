Function Reference

MouseWheel

マウスホイールを上下させます。NT/2000/XP のみ対応。

MouseWheel ( "direction" [, clicks] )

 

パラメータ

direction "up" または "down"
clicks [オプション] ホイールを動かす回数。デフォルトは1。

 

返し値

成功 1を返します。
失敗 directionパラメータを認識できなかった場合0を返します。

 

注意

この関数はNT, 2000, XP以降のOSでのみ動作します。

 

関連

MouseClick, MouseClickDrag, MouseGetPos, MouseMove, MouseCoordMode (Option)

 

例


; ホイールを上に10回動かします。
MouseWheel("up", 10)
