Function Reference

StringLower

文字列を小文字に変換します。

StringLower ( "string" )

 

パラメータ

string 変換する文字列

 

返し値

小文字に変換した文字列

 

注意

なし。

 

関連

StringIsLower, StringIsUpper, StringInStr, StringLeft, StringLen, StringMid, StringRight, StringTrimLeft, StringTrimRight, StringUpper, StringCompare, StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsSpace, StringIsXDigit, StringReplace, StringSplit

 

例


$var = StringLower("I Am A String")
MsgBox(0, "String converted to lowercase is:", $var)
