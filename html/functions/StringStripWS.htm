Function Reference

StringStripWS

文字列からホワイトスペースを取り除きます。

StringStripWS ( "string", flag )

 

パラメータ

string 対象となる文字列
flag 取り除き方を指定するフラグ(足し合わせることで組み合わせることが可能)：
1 = 文頭のホワイトスペース
2 = 文末のホワイトスペース
4 = 文字間の複数のスペース
8 = 全てのスペース(他のフラグの足し合わせと同様)

 

返し値

指定したホワイトスペースを取り除いた新しい文字列が返されます。

 

注意

ホワイトスペースにはChr(9)からChr(13)までの水平タブ、ラインフィード、垂直タブ、フォームフィード、キャリッジリターンが含まれます。 またホワイトスペースにはヌル文字(Chr(0))と標準のスペース(Chr(32))も含まれます。

文字間の単独のスペースを取り除くにはStringReplace関数を使用してください。

 

関連

StringStripCR, StringIsSpace, StringReplace

 

例


$text = StringStripWS("   this    is   a   line    of   text   ", 3)
MsgBox(0, "Stripped from both ends", $text)
