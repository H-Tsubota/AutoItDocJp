Function Reference

FileDelete

ファイルを削除します。

FileDelete ( "path" )

 

パラメータ

Path 削除するファイル(1つ以上)のパス。ワイルドカードを使用可能。

 

返し値

成功 Return 1.
失敗 ファイルが削除できない場合、ファイルが存在しない場合0を返します。

 

注意

注意: FileDeleteに渡された"path"がフォルダの場合、*.*マークを使った場合と同様にその中にあるファイルは削除されます。
ワイルドカードについてはFileFindFirstFileを参照。

ファイル属性によっては削除できないことがあります。

 

関連

FileCopy, FileMove, FileRecycle, DirRemove, FileRecycleEmpty

 

例


FileDelete("D:\*.tmp")
