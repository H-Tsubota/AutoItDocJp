Function Reference

GUICtrlSetCursor

特定のコントロールにマウスカーソルアイコンを設定します。

GUICtrlSetCursor ( controlID, cursorID )

 

パラメータ

controlID GUICtrlCreate...関数によって返されるコントロール識別子(コントロールID)。
cursorID WindowsAPIのSetCursorによって使用されるカーソルID(デフォルトのマウスカーソルを使用する場合-1を使用)。

 

返し値

成功 1を返します。
失敗 0を返します。

 

注意

GUISetCursorがウィンドウ全体でのマウスカーソルを設定するのに対して、この関数はマウスが指定したコントロールにあてられた際に表示されるマウスカーソルを設定します。

カーソルIDが無効な場合、標準の矢印が表示されます。

有効なカーソルIDのリストはMouseGetCursorを参照してください。
CursorId = 16 を設定するとマウスカーソルが非表示になります。

 

関連

GUISetCursor

 

例


#include <GUIConstantsEx.au3>

Opt('MustDeclareVars', 1)

Example()

Func Example()

    GUICreate("put cursor over label", 300, 100)
    GUICtrlCreateLabel("label", 125, 40)
    GUICtrlSetCursor(-1, 4)
    GUISetState()

    While GUIGetMsg() <> $GUI_EVENT_CLOSE
    WEnd
EndFunc   ;==>Example
