Function Reference

Sin

サインを計算します。

Sin ( expression )

 

パラメータ

expression 値（単位はラジアン）

 

返し値

三角関数sinの値を返します

 

注意

1° = pi / 180 rad

 

関連

Cos, Tan, ASin, ACos, ATan

 

例


$pi = 3.14159265358979
$x = sin($pi / 4)  ;0.707106781186547を返します。これはsqrt(2)/2と同じ値です。

$degToRad = $pi / 180
$y = Sin(90 * $degToRad)  ;sin(90°)
