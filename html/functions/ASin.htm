Function Reference

ASin

アークサインを計算します。

ASin ( expression )

 

パラメータ

expression -1 から 1 までの間の値

 

返し値

expressionに対する三角関数arcsinの値を返します。結果はラジアン単位です。

 

注意

ASin(x) は数学的には-1 < x < 1のみに対して定義されています。従ってASinはそれ以外のxに対しては未定義として-1を返します。

 

関連

Sin, Cos, Tan, ACos, ATan

 

例


$x = ASin(0.5)

$pi = 3.14159265358979
$radToDeg = 180 / $pi
$y = ASin(1) * $radToDeg  ;1のアークサインは90°となる。
