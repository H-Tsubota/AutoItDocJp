Function Reference

Beep

ブザー音を鳴らします。

Beep ( [ Frequency [, Duration ]] )

 

パラメータ

Frequency [オプション] ブザー音の周波数(ヘルツ)。37から32,767(0x25から0x7FFF)までの間の任意の数値を指定できます。デフォルトは500Hz。
Duration [オプション] ブザー音の長さ（ミリ秒）。デフォルトは1000ms。

 

返し値

成功、失敗に関わらず1を返します。

 

注意

なし。

 

関連

SoundPlay

 

例


; 500Hzで1秒間ブザー音を鳴らします。

Beep(500, 1000)
