Function Reference

GUICtrlSetDefBkColor

GUIウィンドウの全てのコントロールのデフォルト背景色を設定します。

GUICtrlSetDefBkColor ( defbkcolor [, winhandle] )

 

パラメータ

defbkcolor 全てのコントロールのデフォルト背景色
winhandle [オプション] GUICreateによって返されるウィンドウハンドル(デフォルトでは直前に使用されたウィンドウ)

 

返し値

成功 1を返します。
失敗 0を返します。

 

注意

なし。

 

関連

GUICreate, GUICtrlSetDefColor, GUICtrlSetBkColor

 

例


#include <GUIConstantsEx.au3>

Opt('MustDeclareVars', 1)

Example()

Func Example()
    Local $msg
   
    GUICreate("test GUISetTextColor", 100,100)  ; 中央に表示されるダイアログボックスを作成する

    GUICtrlSetDefBkColor(0xFF0000)  ; 定義される全てのコントロールのテキスト色を変更
   
    GUICtrlCreateLabel("label", 10,5)

    GUICtrlCreateRadio("radio", 10,25,50)
    GUICtrlSetBkColor(-1, 0x0000FF)  ; 指定されたコントロールのテキスト色を変更
   
    GUICtrlCreateButton("button", 10,55)
   
    GUISetState()       ; 空のダイアログボックスを表示

    ; ダイアログボックスが閉じられるまでGUIを実行
    While 1
        $msg = GUIGetMsg()
       
        If $msg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd
EndFunc   ;==>Example
