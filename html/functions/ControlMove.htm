Function Reference

ControlMove

ウィンドウ内でコントロールを動かします。

ControlMove ( "title", "text", controlID, x, y [, width [, height]] )

 

パラメータ

title 対象となるウィンドウ
text 対象となるウィンドウのテキスト
controlID 対象となるコントロール。Controls参照。
x ウィンドウのクライアント領域に対する相対座標での移動先のX座標
y ウィンドウのクライアント領域に対する相対座標での移動先のY座標
width [オプション] 変更後のコントロールの幅
height [オプション] 変更後のコントロールの高さ

 

返し値

成功 1を返します。
失敗 ウィンドウ/コントロールが見つからなかった場合0を返します。

 

注意

xとyがDefaultキーワードの場合移動は起きず、サイズ変更のみがおこなわれます。

 

関連

ControlCommand, ControlGetPos

 

例


ControlMove("[CLASS:Notepad]", "", "MDIClient1", 0, 0, 200, 200)
