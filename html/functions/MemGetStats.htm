Function Reference

MemGetStats

メモリ関連の情報を取得します。

MemGetStats ( )

 

パラメータ

なし。

 

返し値

メモリ情報が格納された7つの要素を持つ配列を返します。

$array[0] = メモリ負荷 (メモリの使用率)
$array[1] = 総物理RAM
$array[2] = 利用可能な物理RAM
$array[3] = 総ページファイル
$array[4] = 利用可能なページファイル
$array[5] = 総仮想メモリ
$array[6] = 利用可能な仮想メモリ

メモリサイズは全てキロバイト単位です。

 

注意

なし。

 

関連

なし。

 

例


$mem = MemGetStats()
MsgBox(0, "Total physical RAM (KB):", $mem[1])
