Function Reference

Eval

文字列によって定義される変数の値を返します。

Eval ( string )

 

パラメータ

string 変数の名前を表す文字列。

 

返し値

成功 変数の値を返します。
失敗 "" (空文字列)を返し、@errorを0以外に設定します。

 

注意

変数の読み込みにEval()を使用する必要があるときは、 多くの場合変数の作成/書き込みにAssign()を、変数の存在確認にIsDeclared()を使用する必要があります。

 

関連

IsDeclared, Assign, Execute

 

例


Dim $a_b = 12
$s = Eval("a" & "_" & "b")  ; $s に12が設定されます。

$s =Eval("c")  ; $s = "" かつ @error = 1
