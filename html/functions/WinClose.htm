Function Reference

WinClose

ウィンドウを閉じます。

WinClose ( "title" [, "text"] )

 

パラメータ

title 閉じるウィンドウの名前。ウィンドウ名の特別な記述の仕方参照
text [オプション]閉じるウィンドウのテキスト

 

返し値

成功 1を返します。
失敗 ウィンドウが見つからなかった場合0を返します。

 

注意

この関数はウィンドウにクローズメッセージを送ります。結果はウィンドウによって異なります（データのセーブをたずねてくる、など）。確実にウィンドウを閉じたい場合はWinKill関数を使用してください。もし複数のウィンドウが閉じる対象に該当する場合、もっともアクティブなものが閉じられます。

 

関連

WinActivate, WinExists, WinKill, WinSetState, WinTitleMatchMode (Option), ProcessClose, WinMove

 

例


WinClose("[CLASS:Notepad]", "")
