Function Reference

Tan

タンジェントを計算します。

Tan ( expression )

 

パラメータ

expression 値（単位はラジアン）

 

返し値

三角関数tanの値を返します

 

注意

1° = pi / 180 rad
Tanは... -3PI/2、 -PI/2、 PI/2、 3PI/2、 5PI/2、 ...で”無限大”になり、このような円周率/2の倍数に対しては1.63317787283838e+016を返すことが多いです。

 

関連

Sin, Cos, ASin, ACos, ATan

 

例


$pi = 3.14159265358979
$x = tan($pi / 4)

$degToRad = $pi / 180
$y = Tan(90 * $degToRad)  ;tan(90°)
