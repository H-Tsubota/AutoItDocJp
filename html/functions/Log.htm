Function Reference

Log

自然対数を計算します。

Log ( expression )

 

パラメータ

expression 正の数値

 

返し値

成功 パラメータで与えられた値の自然対数を返します。
失敗 正でないパラメータには自然対数が定義されていないので-1を返します。

 

注意

@errorは常に0です。

 

関連

Exp

 

例


$x = Log(1000)   ; 6.90775527898214を返します。

$y = Log10(1000) ; 3を返します。

; 常用対数を返すユーザー定義関数
Func Log10($x)
    Return Log($x) / Log(10)  ;10が底
EndFunc
