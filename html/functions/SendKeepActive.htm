Function Reference

SendKeepActive

Send()の間、指定されたウィンドウをアクティブな状態にするようにします。

SendKeepActive ( "title" [, "text"] )

 

パラメータ

title アクティブ化するウィンドウのタイトル。ウィンドウ名の特別な記述の仕方を参照。この関数を無効にするには空タイトルを使用してください。
text [オプション] ウィンドウのテキスト

 

返し値

成功 1を返します。
失敗 ウィンドウが見つからない場合、0を返します。

 

注意

Send()による各キーストロークのシミュレーション間にアクティブウィンドウをリセットしようとします。

 

関連

Send

 

例


Run("notepad.exe")
WinWait("[CLASS:Notepad]")

SendKeepActive("[CLASS:Notepad]")

; 一時停止の間にアクティブなウィンドウを変更
For $i = 1 to 10
    Sleep(1000)
    Send("Hello")
