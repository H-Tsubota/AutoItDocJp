Function Reference

ChrW

UNICODEのコードに対応する文字を返します。

ChrW ( UNICODEcode )

 

パラメータ

UNICODEcode 0-65535の範囲にあるUNICODEのコード (例えば65は大文字のAを返します)。

 

返し値

成功 指定されたコードが表す文字を格納した文字列を返します。
失敗 UNICODEcodeが65535より多きい場合、空文字列を返して、@errorを1に設定します。

 

注意

完全なASCIIコードの表については付録を参照。

 

関連

Asc, AscW, Chr, String

 

例


$text = ""
For $i = 256 to 512
    $text = $text & ChrW($i)
Next
MsgBox(0, "Unicode chars 256 to 512", $text)
