Function Reference

GUICtrlSetBkColor

コントロールの背景色を設定します。

GUICtrlSetBkColor ( controlID, backgroundcolor )

 

パラメータ

controlID GUICtrlCreate...関数によって返されるコントロール識別子(コントロールID)。
backgroundcolor 使用するRGB色。

 

返し値

成功 1を返します。
失敗 0を返します。

 

注意

コントロールのうちボタン、ラベル、チェックボックス、グループ、ラジオ、エディット、インプット、リスト、リストビュー、リストビューアイテム、ツリービュー、ツリービューアイテム、グラフィック、プログレス、スライダー、コンボのみ色設定できます。

"Windows XP style"が使用されている場合、プログレスコントロールは塗りつぶしできません。

ボタンコントロールは常に"Windows Classic style"で塗りつぶされます。$BS_ICONスタイルを設定することはできません。

特殊フラグの$GUI_BKCOLOR_TRANSPARENTをラベルコントロール、グループコントロール、ラジオコントロール、チェックボックスコントロールに使用することで透明背景を設定できます。

特殊フラグの$GUI_BKCOLOR_TRANSPARENTをリストビューコントロールに使用することで、リストビューアイテムのラインの背景色を交互にかわるように設定できます。
奇数番目のラインはGUICtrlSetBkColorでリストビューコントロールに設定された色になります。
偶数番目のラインはGUICtrlSetBkColorでリストビューアイテムコントロールに設定された色になります。

 

関連

GUICtrlCreate..., GUICtrlSetColor, GUICtrlSetDefBkColor>

 

例


#include <GUIConstantsEx.au3>

Opt('MustDeclareVars', 1)

Example()

Func Example()
    Local $msg
   
    GUICreate("My GUI background color")  ; 表示時に中央に表示されるダイアログボックスを作成

    GUICtrlCreateLabel("my label", 10, 20)
    GUICtrlSetBkColor(-1, 0x00ff00)     ; 緑
   
    GUISetState()

    ; ダイアログが閉じられるまでGUIを実行
    While 1
        $msg = GUIGetMsg()
       
        If $msg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd
EndFunc   ;==>Example
