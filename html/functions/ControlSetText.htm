Function Reference

ControlSetText

コントロールのテキストを設定します。

ControlSetText ( "title", "text", controlID, "new text" [, flag] )

 

パラメータ

title 対象となるウィンドウの名前
text 対象となるウィンドウのテキスト
controlID 操作するコントロール。Controls参照
new text コントロールに新しく設定するテキスト
flag [オプション] 0 （デフォルト）以外の値を設定するとターゲットウィンドウに再描画をさせます。

 

返し値

成功 1を返します。
失敗 ウィンドウ/コントロールが見つからなかった場合0を返します。

 

注意

なし。

 

関連

ControlGetText, ControlCommand, ControlSend, SplashTextOn

 

例


Run("notepad.exe")
WinWait("[CLASS:Notepad]")
ControlSetText("[CLASS:Notepad]", "", "Edit1", "New Text Here" )
