Function Reference

PluginOpen

プラグインファイルを開きます。

PluginOpen ( filename )

 

パラメータ

filename 開くプラグインファイルのファイル名

 

返し値

成功 プラグインを閉じる時に使用する"ハンドル"を返します。
失敗 0を返します。

 

注意

なし。

 

関連

PluginClose

 

例


$handle = PluginOpen("example.dll")
