Function Reference

HttpSetProxy

http接続に使用するインターネットプロキシを設定します。

HttpSetProxy ( mode [, "proxy:port" [, "username" [, "password"]]] )

 

パラメータ

mode 使用するプロキシモード
0 = (デフォルト) 現在のインターネットエクスプローラの設定をプロキシに使用する
1 = プロキシを使用しない(ダイレクト接続)。
2 = 指定したプロキシを使用する。
proxy:port [オプション] 使用するプロキシのアドレスとポート。下記、備考を参照。
username [オプション] 必要な場合、プロキシ用ユーザー名
password [オプション] 必要な場合、プロキシ用パスワード

 

返し値

なし。

 

注意

この関数の動作にはインターネットエクスプローラ3以上が必要です。

CERNプロキシサーバーのみサポートされています。

プロキシサーバーは通常ウェブアドレスとポート番号で構成されています。ほとんどのISPはポート番号8080を使用しています。あなたのプロクシが"www.proxy.com"、ポート番号"8080"の場合、次のようにしてプロキシを設定します。
HttpSetProxy(2, "www.proxy.com:8080")

ほとんどのプロキシサーバーはユーザー名とパスワードを必要としません。

mode=0を使用時にusernameとpasswordを指定した場合、インターネットエクスプローラの設定プロキシで使用されます。

 

関連

InetGet, InetGetSize, InetRead, FtpSetProxy

 

例


; プロキシを使用しない
HttpSetProxy(1)

; IEデフォルトをプロキシに使用
HttpSetProxy(0)

; "www-cache.myisp.net" ポート8080をプロキシに使用
HttpSetProxy(2, "www-cache.myisp.net:8080")
