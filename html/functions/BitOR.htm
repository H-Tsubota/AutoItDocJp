Function Reference

BitOR

ビット演算のOR演算子として働きます。

BitOR ( value1, value2 [, value n] )

 

パラメータ

value1 １番目の数
value2 2番目の数
value n [オプション] n番目の数。255個まで値を指定できます。

 

返し値

OR演算の結果を返します。
ビット演算は32ビット整数としておこなわれます。

 

注意

16進数表現を使用することができます。
BitANDは全ての入力引数が0のビット位置で対応する位置に0を返し、1のビットが1つでもある場合は1を返します。

 

関連

BitAND, BitNOT, BitShift, BitXOR, Hex, BitRotate

 

例


$x = BitOR(3, 6) ; 0011 OR 0110 = 0111 なので x == 7

$x = BitOR(3, 15, 32) ; 0011 OR 1111 OR 00100000 = 00101111 なので x == 47
