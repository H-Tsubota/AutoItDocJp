Function Reference

GUICtrlSetColor

コントロールのテキスト色を設定します。

GUICtrlSetColor ( controlID, textcolor)

 

パラメータ

controlID GUICtrlCreate...関数によって返されるコントロール識別子(コントロールID)。
backgroundcolor 使用するRGB色。

 

返し値

成功 1を返します。
失敗 0を返します。

 

注意

コントロールのうちボタン、ラベル、チェックボックス、グループ、ラジオ、エディット、インプット、リスト、リストビュー、リストビューアイテム、ツリービュー、ツリービューアイテム、グラフィック、プログレス、コンボのみ色設定できます。

"Windows XP/Vista style"が使用されている場合、チェックボックスコントロール、ラジオコントロール、グループコントロール、プログレスコントロールは塗りつぶしできません。

ボタンコントロールは常に"Windows Classic style"で塗りつぶされます。

 

関連

GUICtrlCreate..., GUICtrlSetBkColor, GUICtrlSetDefColor

 

例


#include <GUIConstantsEx.au3>

Opt('MustDeclareVars', 1)

Example()

Func Example()
    Local $msg
   
    GUICreate("My GUI color text")  ; 表示時に中央に表示されるダイアログボックスを作成

    GUICtrlCreateLabel("my Red label", 10, 20)
    GUICtrlSetColor(-1, 0xff0000)   ; 赤
   
    GUISetState()

    ; ダイアログが閉じられるまでGUIを実行
    While 1
        $msg = GUIGetMsg()
       
        If $msg = $GUI_EVENT_CLOSE Then ExitLoop
    WEnd
EndFunc   ;==>Example
