Function Reference

Number

値の数値表現を返します。

Number ( expression )

 

パラメータ

expression 値の数値表現を返します。

 

返し値

数値が返されます。

 

注意

数字でない文字で始まる文字列は数値0となります。 数字で始まる文字列は数字以外の文字が取り除かれます。

 

関連

Int, String, Binary, Ceiling, Floor, HWnd, Ptr, Round

 

例


$w = Number(1+2+10) ;13を返す
$x = Number("3.14") ;3.14を返す
$y = Number("24/7") ;24を返す
$z = Number("tmp3") ;0を返す
