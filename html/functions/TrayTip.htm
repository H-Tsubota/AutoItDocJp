Function Reference

TrayTip

AutoItアイコンからバルーンチップを表示します(2000/XPのみ)

TrayTip ( "title", "text", timeout [, option] )

 

パラメータ

title バルーンチップの最上段に太文字で表示されるテキスト(最大63文字)
text バルーンチップに表示されるメッセージ(最大255文字)
timeout バルーンチップが表示されているおおよその時間(単位：秒)(Windowsでは10-30秒の範囲で最大最大値を設定できますが、この設定は必ずしも正確に動作しません)。
option [オプション] 注意を参照。 0=アイコンなし (デフォルト),
1=情報アイコン, 2=警告アイコン, 3=エラーアイコン

 

返し値

なし。

 

注意

TrayTipはWindows2000/XP以降でのみ動作します。

TrayTip はAutoItトレイアイコンが消えると動作しません。従って、TrayTipはAutoItSetOption("TrayIconHide", 1)が設定されている場合、ユーザーがレジストリ設定でバルーンチップを無効にしている場合、動作しません!

WindowsXPは通常、バルーンチップ表示時に音を鳴らします。音を消す場合はoptionsパラメータに16を足すか、空文字列をtitleに設定します。

表示されているバルーンチップを消す場合は空文字列をtextパラメータに設定して(任意のtitleで)再度この関数を呼び出します。

 

関連

MsgBox, ToolTip, TrayIconHide (Option), TraySetToolTip

 

例


TrayTip("I'm a title", "I'm the message", 5, 1)
MsgBox(4096,"", "Press OK to see another tip.")
TrayTip("clears any tray tip","",0)
TrayTip("", "A different tray tip.", 5)
Sleep(5000)
