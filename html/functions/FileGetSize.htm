Function Reference

FileGetSize

ファイルのサイズをバイト単位で返します。

FileGetSize ( "filename" )

 

パラメータ

filename 調べるファイル名

 

返し値

成功 ファイルのサイズをバイト単位で返します。
失敗 0を返し、@errorを1に設定します。

 

注意

フォルダに対しては動作しません。
キロバイト単位に変換するには1024で、メガバイト単位に変換するには1048576で割ってください。

 

関連

FileGetAttrib, FileGetTime, DriveSpaceTotal, FileGetVersion

 

例


$size = FileGetSize("AutoIt.exe")
