Function Reference

DriveMapDel

ネットワークドライブを切断します。

DriveMapDel ( "drive" )

 

パラメータ

drive 切断するデバイス。例："O:"、"LPT1:"。

 

返し値

成功 1を返します。
失敗 切断に失敗した場合0を返します。

 

注意

マップされているドライブ文字がない場合、接続名(例：\\server\share)を切断に使用できます。

 

関連

DriveMapAdd, DriveMapGet

 

例



; ドライブXを\\myserver\stuffに現在のユーザーでマップ
DriveMapAdd("X:", "\\myserver\stuff")

; 切断
DriveMapDel("X:")
