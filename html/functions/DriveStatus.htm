Function Reference

DriveStatus

ドライブの状態を文字列で返します。

DriveStatus ( "path" )

 

パラメータ

path 調べたいドライブのパス

 

返し値

値 意味
UNKNOWN ドライブが未フォーマット(RAW)
READY リムーバルメディアを接続しているドライブ/ドライバなど
NOTREADY メディアを挿入していないフロッピー、CDドライブなど
INVALID 指定したドライブ文字が存在しない、またはマップしたネットワークドライブにアクセスできない。

 

注意

返し値のリストにない値が返される場合があります。
DriveStatusは完全なディレクトリパスに対しても(ディレクトリが存在すれば)動作します。ただしファイルパスに対しては動作しません。

 

関連

DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, CDTray, FileExists

 

例


$var = DriveStatus( "c:\" )
MsgBox(4096,"Status",$var)
