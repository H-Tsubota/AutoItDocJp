Function Reference

DriveSpaceTotal

パスの総ディスク容量をメガバイト単位で返します。

DriveSpaceTotal ( "path" )

 

パラメータ

path 調べたいドライブのパス

 

返し値

成功 ディスク容量をメガバイト単位の浮動小数点で返します。
失敗 @errorを1に設定します。

 

注意

DriveSpaceTotalは完全なディレクトリパスに対しても(ディレクトリが存在すれば)動作します。ただしファイルパスに対しては動作しません。

 

関連

DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetSerial, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveStatus, FileGetSize

 

例


$var = DriveSpaceTotal( "c:\" )
MsgBox(4096, "Total Space on C:", $var & " MB")
