Function Reference

StringTrimRight

文字列の右側から指定文字分だけ取り除きます。

StringTrimRight ( "string", count )

 

パラメータ

string 対象となる文字列
count 取り除く文字数

 

返し値

右側からcount文字分だけ取り除いた文字列が返されます。

 

注意

countが不正な値の場合、空の文字列が返されます。
StringTrimRight($str, $n)はStringLeft($str, StringLen($str) - $n)と同値です。

 

関連

StringInStr, StringLeft, StringLen, StringLower, StringMid, StringRight, StringTrimLeft, StringUpper, StringCompare, StringReplace, StringSplit

 

例


$result = StringTrimRight("I am a string", 3)
MsgBox(0, "String without rightmost 3 characters is:", $result)
