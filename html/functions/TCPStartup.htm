Function Reference

TCPStartup, UDPStartup

TCPサービスまたはUDPサービスを開始します。

TCPStartup ( )
UDPStartup ( )

 

パラメータ

なし。

 

返し値

成功 1を返します。
Failure: 0を返し、@errorを設定します。
@error: WindowsAPIのWSAStartupの返し値 (see MSDNを参照)。

 

注意

メモリ消費を抑えるためにはTCPShutdown()の呼び出しが必要です。
スクリプトは成功した各TCPStartup()呼び出しごとにTCPShutdown()を呼ぶ必要があります。
UDPStartup()は単にTCPStartup()を呼び代えたものです。

 

関連

TCPShutdown, TCPListen, TCPCloseSocket, UDPCloseSocket, TCPAccept, TCPConnect, TCPNameToIP, TCPRecv, TCPSend

 

例


TCPStartup ( )  ; TCPサービス開始
