Function Reference

Asc

文字のASCIIコードを返します。

Asc ( "char" )

 

パラメータ

char コードを知りたい文字。文字列が与えられた場合、最初の文字が使われます。

 

返し値

指定した文字のASCIIコードを返します。

 

注意

ASCIIコードの一覧は付録を参照。
Asc("0") == 48, Asc("9") == 57, Asc("A") == 65, Asc("Z") == 90, Asc("a") == 97, Asc("z") == 122, etc.

 

関連

AscW, Chr, ChrW

 

例


$code = Asc("A")
MsgBox(0, "ASCII code for A:", $code)
