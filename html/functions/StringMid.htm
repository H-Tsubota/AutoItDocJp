Function Reference

StringMid

文字列から指定した文字数分を取得します。

StringMid ( "string", start [, count] )

 

パラメータ

string 対象となる文字列
start 取得を開始する文字の位置(1 = 1文字目)。
count [オプション] 取得する文字数。デフォルトは開始位置以降の文字列全て。

 

返し値

取得した文字列を返します。

 

注意

startパラメータが適正な範囲をこえている場合、空の文字列が返されます。startパラメータが適正でcountパラメータが適正な範囲をこえている場合、開始位置以降の文字列全てが返されます。

 

関連

StringInStr, StringLeft, StringLen, StringLower, StringRight, StringTrimLeft, StringTrimRight, StringUpper, StringCompare, StringReplace, StringSplit

 

例


$var = StringMid("I am a string", 3, 2)
MsgBox(0, "2 chars extracted from position 3 are:", $var)
