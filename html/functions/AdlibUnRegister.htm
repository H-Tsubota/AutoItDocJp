Function Reference

AdlibUnRegister

アドリブ関数の登録を解除します。

AdlibUnRegister( [ "function" ] )

 

パラメータ

function [オプション] 登録解除するアドリブ関数の名前。詳しい情報については注意を参照。

 

返し値

 

注意

関数名を指定しなかった場合、最後に登録された関数が登録解除されます。

 

関連

AdlibRegister

 

例


AdlibRegister("MyAdlib")
;...
AdlibUnRegister("MyAdlib")

Func MyAdlib()
    ;... 実行はブロックされてはならない。...Wait(), MsgBox(), InputBox()関数は避ける
EndFunc
