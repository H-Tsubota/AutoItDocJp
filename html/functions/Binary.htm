Function Reference

Binary

式のバイナリ表現を返します。

Binary ( expression )

 

パラメータ

expression バイナリ/バイトデータに変換する式

 

返し値

バイナリ変数を返します。

 

注意

なし。

 

関連

Int, Number, IsBinary, BinaryLen, BinaryMid, BinaryToString, StringToBinary

 

例


$var = Binary("0x00204060")
;$varはバイナリ型
msgbox(0, "IsBinary", IsBinary($var))
