Function Reference

TimerDiff

以前のTimerInit()の呼び出しからの経過時間を返します。

TimerDiff ( timestamp )

 

パラメータ

timestamp 以前に呼び出されたTimerInit()が返したタイムスタンプ

 

返し値

以前のTimerInit()の呼び出しからの経過時間を（ミリ秒で）返します。

 

注意

なし。

 

関連

TimerInit

 

例


$begin = TimerInit()
sleep(3000)
$dif = TimerDiff($begin)
MsgBox(0,"Time Difference",$dif)
