Function Reference

DriveGetSerial

ドライブのシリアルナンバーを返します。

DriveGetSerial ( "path" )

 

パラメータ

path 調べたいドライブのパス

 

返し値

成功 ドライブのシリアルナンバーを文字列で返します。
失敗 @errorを１に設定します。

 

注意

返し値はドライブのラベル上に見られるハードウェアのシリアルナンバーではなくドライブのWindowsボリュームIDです。

 

関連

DriveGetDrive, DriveGetFileSystem, DriveGetLabel, DriveGetType, DriveSetLabel, DriveSpaceFree, DriveSpaceTotal, DriveStatus

 

例


$var = DriveGetSerial( "c:\" )
MsgBox(4096, "Serial Number: ", $var)
