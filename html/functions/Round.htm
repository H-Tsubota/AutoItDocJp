Function Reference

Round

指定した桁で丸めた数を返します。

Round ( expression [, decimalplaces] )

 

パラメータ

expression 適切な数値表現
decimalplaces [オプション] 小数点何桁目で丸めこむかを指定する数。負の値を指定した場合、Roundは整数を返します。

 

返し値

丸めた値を返します。

 

注意

1の位、10の位、100の位、・・・などの位で丸める場合、decimalplacesパラメータに負の値を指定します。 15桁まで表示されます。また decimalplacesで指定した桁以下の0は丸めたあと切り捨てられます。

 

関連

Int, Number, Ceiling, Floor, Random

 

例


$x = Round(-1.582, 1)  ;-1.6を返します。
$y = Round(3.1415, 9)  ;変化なし。
$z = Round(123.5, -1)  ;120を返します。
