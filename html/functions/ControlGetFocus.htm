Function Reference

ControlGetFocus

指定したウィンドウ内でキーボードフォーカスのあっているコントロールのControlRef#を返します。

ControlGetFocus ( "title" [, "text"] )

 

パラメータ

title 調べるウィンドウの名前
text [オプション] 調べるウィンドウのテキスト

 

返し値

成功 指定したウィンドウ内でキーボードフォーカスのあっているコントロールのClassNameNNを返します。
失敗 ウィンドウが見つからなかった場合 "" (空の文字列)を返し、@errorを1に設定します。

 

注意

なし。

 

関連

ControlFocus, ControlCommand

 

例


Run("notepad.exe")
$a = ControlGetFocus("[CLASS:Notepad]")
