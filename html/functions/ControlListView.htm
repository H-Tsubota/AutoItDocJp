Function Reference

ControlListView

ListView32 コントロールにコマンドを送信します。

ControlListView ( "title", "text", controlID, "command" [, option1 [, option2]] )

 

パラメータ

title アクセスするウィンドウの名前
text アクセスするウィンドウのテキスト
controlID 対象となるコントロール。Controls参照。
command コントロールに送信するコマンド（下表参照）
option1 [オプション] 一部のコマンドで必要となる追加パラメータ
option2 [オプション] .一部のコマンドで必要となる追加パラメータ

 

返し値

下表に示すコマンドによって異なる。 エラーの場合（コマンドの間違いやウィンドウ/コントロールが見つからなかった場合）は@errorを1に設定します。


コマンド、オプション1、オプション2 処理
"DeSelect", From [, To] 1つまたはそれ以上のアイテムの選択を解除します。
"FindItem", "string to find" [, SubItem] 指定した文字列のアイテムのアイテム番号を返します。見つからなかった場合-1を返します。
"GetItemCount" アイテムの数を返します。
"GetSelected" [, option] 選択されているアイテムのアイテム番号を格納した文字列を返します。option=0 (デフォルト) では一番最初のアイテムの番号のみ返されます。option=1 ではすべての選択されたアイテムのアイテム番号が｜で区切られて（例："0|3|4|10"）返されます。アイテムが選択されていない場合、空の文字列""が返されます。
"GetSelectedCount" 選択されているアイテムの数を返します。
"GetSubItemCount" サブアイテムの数を返します。
"GetText", Item, SubItem アイテム/サブアイテムで指定した位置のテキストを返します。
"IsSelected", Item 指定したアイテムが選択されている場合1を返し、それ以外で0を返します。
"Select", From [, To] 1つまたはそれ以上のアイテムを選択します。
"SelectAll" 全てのアイテムを選択します。
"SelectClear" 全てのアイテムの選択を解除します。
"SelectInvert" 現在の選択状態を反転させます。
"ViewChange", "view" 現在の表示状態を切り替えます。有効な表示状態は"list", "details", "smallicons", "largeicons"です。


すべてのアイテム/サブアイテムは0を基準とします。つまり、リストの最初のアイテム/サブアイテムが0、2番目が1、・・・となります。
ListView32 コントロールの"詳細"表示では、"アイテム"は"行"で、"サブアイテム" は"列"と考えることができます。

 

注意

32ビットのAutoItプロセスを使って64ビットのプロセスから読み込みをするとコマンドが失敗することがあります。同様に64ビットのAutoItプロセスを使って32ビットのプロセスから読み込みをするとコマンドが失敗することがあります。

 

関連

ControlCommand

 

例


ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "SelectAll")
ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "Deselect", 2, 5)
MsgBox(0, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "GetText", 9, 0) )
MsgBox(0, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "FindItem", "14 KB", 1) )
MsgBox(0, "", ControlListView(@ProgramFilesDir & "\AutoIt3", "", "SysListView321", "GetSelected", 1) )
