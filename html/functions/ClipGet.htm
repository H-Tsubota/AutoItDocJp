Function Reference

ClipGet

クリップボードからテキストを取得します。

ClipGet ( )

 

パラメータ

なし。

 

返し値

成功 クリップボード上のテキストを文字列として返します。
失敗 クリップボードが空の場合@errorを1に設定します。
テキストでない内容の場合2に設定します。
クリップボードにアクセスできない場合、3または4を返します。

 

注意

複数選択されたファイル/ディレクトリがクリップボード上にある場合、ファイル名/ディレクトリ名が@LFで区切られたテキストとして返されます。

 

関連

ClipPut

 

例


$bak = ClipGet()
MsgBox(0, "Clipboard contains:", $bak)

ClipPut($bak & "additional text")
MsgBox(0, "Clipboard contains:", ClipGet())
