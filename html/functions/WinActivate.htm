Function Reference

WinActivate

ウィンドウをアクティブにします（ウィンドウをフォーカスします）。

WinActivate ( "title" [, "text"] )

 

パラメータ

title アクティブにするウィンドウの名前。ウィンドウ名の特別な記述の仕方参照
text [オプション] アクティブにするウィンドウのテキスト

 

返し値

成功 ウィンドウのハンドルを返します。
失敗 ウィンドウが見つからなかった場合またはアクティブにできなかった場合、0を返します。

 

注意

WinActiveを使えばＷｉｎＡｃｔｉｖａｔｅが成功したかどうか確認することができます。もし複数のウィンドウが処理の対象に該当する場合、もっともアクティブなウィンドウがアクティブ化されます。WinActivateは最小化されたウィンドウにも有効です。しかし”常にトップ”に設定されたウィンドウより前にだすことはできません。

@extendedにはどのようにしてプロセスがアクティブ化されたかについての追加情報が格納されます。

 

関連

WinClose, WinSetState, WinTitleMatchMode (Option), WinKill, WinMove

 

例


WinActivate("[CLASS:Notepad]", "")
