Function Reference

StringIsDigit

文字列が数字(0-9)のみからなるかどうか調べます。

StringIsDigit ( "string" )

 

パラメータ

string 調べる文字列

 

返し値

成功 1を返します。
失敗 文字列が数字以外の文字を含む場合0を返します。

 

注意

例を参照

 

関連

StringIsAlNum, StringIsAlpha, StringIsASCII, StringIsLower, StringIsSpace, StringIsUpper, StringIsXDigit, StringIsInt

 

例


StringIsDigit("12333")  ;1を返す
StringIsDigit("1.5")    ;小数点があるため0を返す
StringIsDigit("1 2 3")  ;スペースがあるため0を返す
StringIsDigit("")       ;0を返す
