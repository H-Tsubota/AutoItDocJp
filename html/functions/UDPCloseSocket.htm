Function Reference

UDPCloseSocket

UDPソケットを閉じます。

UDPCloseSocket ( socketarray )

 

パラメータ

socketarray UDPBind、UDPOpen関数によって返されるソケット/配列。

 

返し値

成功 1を返します。
失敗 0を返し、@errorを設定します。
@error: 無効なsocketarrayの場合、-1、-2または-3 。
WindowsAPIのWSAGetErrorの返し値(MSDNを参照)

 

注意

なし。

 

関連

UDPBind, UDPOpen, TCPShutdown, TCPStartup

 

例


;サーバー!! 最初に実行!!!!!!!!!!!!!!!
$g_IP = "127.0.0.1"

; UDPサービスを開始
;==============================================
UDPStartUp()

; クリーンアップ関数を登録
OnAutoItExitRegister("Cleanup")

; リスニング用"ソケット"を作成
;==============================================
$socket = UDPBind($g_IP, 65432)
If @error <> 0 Then Exit

;--- 行いたい処理をここに書く

Func Cleanup()
    UDPCloseSocket($socket)
    UDPShutdown()
EndFunc
