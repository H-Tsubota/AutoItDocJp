Function Reference

IsInt

変数または式が整数型かどうかを調べます。

IsInt ( variable )

 

パラメータ

variable 調べる変数/式.

 

返し値

成功 1を返します。
失敗 整数で無い場合0を返します。

 

注意

IsInt(7.5 - 2.5) は 1(真)を返します。

 

関連

IsArray, IsFloat, IsString, IsNumber, IsBool, IsHWnd, StringIsInt, IsBinary, IsPtr, VarGetType

 

例


IsInt(-12345)     ;1を返します。
IsInt(3.0000)     ;1を返します。
IsInt("5432")     ;文字列なので0を返します。
IsInt(7.5 - 4.5)  ;整数の3なので1を返します。
