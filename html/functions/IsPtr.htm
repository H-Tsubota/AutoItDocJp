Function Reference

IsPtr

変数の基本型がポインタかどうか調べます。

IsPtr ( variable )

 

パラメータ

variable 調べる変数/式

 

返し値

成功 式がポインタ型の場合、1を返します。
失敗 式がポインタ型でないの場合、0を返します。

 

注意


 

関連

IsArray, IsFloat, IsHWnd, IsInt, IsString, IsNumber, IsBool, Ptr

 

例


Run("notepad.exe")
Local $hWnd = WinGetHandle("[CLASS:Notepad]")
If IsPtr($hWnd) Then
    MsgBox(4096, "", "It's a valid Ptr")
Else
    MsgBox(4096, "", "It's not a valid Ptr")
EndIf
