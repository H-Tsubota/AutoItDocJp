Function Reference

TCPShutdown, UDPShutdown

TCP/UDPサービスを停止します。

TCPShutdown ( )
UDPShutdown ( )

 

パラメータ

なし。

 

返し値

成功 1を返します。
失敗 0を返し、@errorを設定します。
@error: WindowsAPIのWSACleanupの返し値(MSDNを参照)。

 

注意

スクリプトはTCPStartup()の呼び出しが成功するごとにTCPShutdown()を一回、呼び出す必要があります。
UDPShutdown()は単にTCPShutdown()を呼び代えたものです。

 

関連

TCPStartup, TCPListen, TCPCloseSocket, UDPCloseSocket

 

例


TCPStartup()    ; TCPサービス開始

TCPShutdown ( ) ; TCPサービス停止
