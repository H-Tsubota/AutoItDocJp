Function Reference

AutoItWinSetTitle

AutoItウィンドウのタイトルを変更します。

AutoItWinSetTitle ( "newtitle" )

 

パラメータ

newtitle 新しくウィンドウにつけるタイトル

 

返し値

なし。

 

注意

AutoItウィンドウは通常は隠されています。タイトルを変更する目的は他のプログラム(もしくは他のAutoItスクリプト)との連携を可能にすることです。

 

関連

AutoItWinGetTitle, WinSetTitle

 

例


AutoItWinSetTitle("My AutoIt Window")
