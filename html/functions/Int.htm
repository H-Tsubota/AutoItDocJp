Function Reference

Int

値の整数(全数)表現を返します。

Int ( expression )

 

パラメータ

expression 整数に変換される値

 

返し値

成功 整数が返されます。
失敗 整数でも浮動小数でも文字列でもない場合0が返され、@errorが1に設定されます。

 

注意

小数部分は切り捨てられます。たとえばInt(1.999999)は1を返します。
Int(0/0) は -9223372036854775807を返します。
この関数は浮動小数点数の不正確性を考慮して浮動小数点数に対して小規模な修正をおこないます。例えば浮動小数点の式 0.7 + 0.2 + 0.1 によって計算される浮動小数点は正確に1.0ではありません。Int()では自動的にこれを修正します。ただし非常にまれな特定の状況ではInt()は予期しない値を返すことがあります(予期しない値が返される確率はInt()が全く修正処理をおこなわない場合に不正確な値が返される確率よりも小さいです)。

 

関連

Number, String, Round, HWnd, Binary, Ceiling, Floor, Ptr

 

例


$var = Int(10.793) ;$varは整数の10
