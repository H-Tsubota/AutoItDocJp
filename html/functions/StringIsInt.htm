Function Reference

StringIsInt

文字列が整数かどうかを調べます。

StringIsInt ( "string" )

 

パラメータ

string 調べる文字列

 

返し値

成功 1を返します。
失敗 文字列が整数で無い場合0を返します。

 

注意

StringIsIntは文字列でない整数式に対しても1を返します。 ただし、StringIsIntは"4ff0"のような16進数の式に対しては0を返します。 許される記号は文字列1文字目のプラス、マイナス記号のみです。

 

関連

StringIsFloat, StringIsDigit, IsInt

 

例


StringIsInt("+42")  ;1を返す
StringIsInt("-00")  ;1を返す
StringIsInt("1.0")  ;小数点が含まれるので0を返す
StringIsInt(1.0)    ;数から文字列へ変換されるので1を返す
StringIsInt("1+2")  ;プラス記号を含むので0を返す
