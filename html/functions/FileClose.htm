Function Reference

FileClose

開いているテキストファイルを閉じます。

FileClose ( filehandle )

 

パラメータ

filehandle FileOpen関数によって取得されたファイルのハンドル。

 

返し値

成功 1を返します。
失敗 無効なファイルハンドルの場合、0を返します。

 

注意

終了時、AutoItは自動的に全ての開いているファイルを閉じますが、FileCloseの使用がより適切です。

この関数はFileFindFirstFile()によって返されるサーチハンドルを閉じる際にも使用されます。

 

関連

FileFindFirstFile, FileOpen, FileFindNextFile, FileFlush

 

例


$handle = FileOpen("test.txt", 0)

FileClose($handle)
