Function Reference

FileRecycle

ファイル、フォルダをゴミ箱に送ります。

FileRecycle ( "source" )

 

パラメータ

source 捨てるファイル、フォルダのパス。ワイルドカード使用可。

 

返し値

成功 1を返します。
失敗 0を返します(基本的にはファイルが使用中か、存在しない場合) 。

 

注意

ワイルドカードについてはFileFindFirstFileを参照。
フォルダを削除する場合は、パスの最後にバックスラッシュをつけないでください。

 

関連

FileDelete, FileRecycleEmpty, DirRemove, FileMove

 

例


FileRecycle("C:\*.tmp")
