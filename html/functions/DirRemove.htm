Function Reference

DirRemove

ディレクトリ/フォルダを削除します。

DirRemove ( "path" [, recurse] )

 

パラメータ

path 削除するフォルダのパス
recurse [オプション] 削除したいフォルダの下にあるサブフォルダを削除するかどうか指定するフラグ
  0 = (デフォルト) ファイルとサブフォルダを削除しません
  1 = ファイルとフォルダを削除します(DOSのDelTreeコマンドと同様)

 

返し値

成功 1を返します。
失敗 フォルダ削除でエラーが起こった場合(またフォルダが存在しなかった場合)0を返します。

 

注意

フォルダの属性によっては削除できません。

 

関連

DirCreate, FileRecycle, DirCopy, DirMove, FileDelete

 

例


; C:\Test1とその下の全てのフォルダ・ファイルを削除します。
DirRemove("C:\Test1", 1)
