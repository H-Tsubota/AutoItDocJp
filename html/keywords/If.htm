Keyword Reference

If...Then

条件つき１行命令文

If <expression> Then statement

 

パラメータ

expression もし条件（expression）が真であれば、命令文（statement）が実行されます。

 

注意

このバージョンのIf文は1行の命令をEndIfの処理に時間をかけずに実行します。
expressionには必要に応じて括弧と共に論理演算子<, <=, >, >=, =, ==, <>とブール演算子AND, OR, NOTを使用することができます。

 

関連

If...Else...EndIf, Select...Case...EndSelect, Switch...EndSwitch

 

例


; コマンドラインからの引値がない場合終了します。
If $CmdLine[0] = 0 Then Exit

;同じ処理:
If $CmdLine[0] = 0 Then
    Exit
EndIf
