Keyword Reference

Default

関数呼び出し時に使用されるキーワード値

    $var = Default

 

パラメータ

なし

 

注意

このキーワードは通常の計算式内で使用できません。AutoItはパフォーマンス上の理由からそのようなエラーを検出しません。

パラメータ渡しの際に使用した場合、その動作は対応するAutoItの関数ドキュメントで指定されたものになります。
UDFの場合、ラメータにDefaultが設定されたかどうか、またその際に希望どおりの動作が行われているかどうかのチェックはスクリプト作成者の責任となります。
使用した場合、渡されたパラメータはDefaultキーワードに設定され、定義されている場合でもオプションパラメータ値にはなりません。

 

関連

IsKeyWord

 

例


WinMove("[active]","",default, default, 200,300)    ; アクティブなウィンドウのサイズ変更をおこなう(移動はしない)

MyFunc2(Default,Default)

Func MyFunc2($Param1 = Default, $Param2 = 'Two', $Param3 = Default)
    If $Param1 = Default Then $Param1 = 'One'
    If $Param3 = Default Then $Param3 = 'Three'

    MsgBox(0, 'Params', '1 = ' & $Param1 & @LF & _
        '2 = ' & $Param2 & @LF & _
        '3 = ' & $Param3)
EndFunc
