Keyword Reference

#RequireAdmin

スクリプトの実行に管理者権限を必要とするように設定します。

#RequireAdmin

 

パラメータ

なし

 

注意

この機能は当初はAutoItがWindowsVistaのユーザーアカウントコントロール(UAC)上でも正常に動作するよう許可することを目的にしていました(しかし Windows 2000、Windows XPでも機能します)。

より詳細な情報についてはAutoIt on Windows Vistaを参照。

この機能で起動した新しいプロセスではConsolewrite()などの一部の関数が動作しません(Sciteで何も表示されません)。

 

関連

IsAdmin

 

例


#RequireAdmin

MsgBox(4096,"Info","Now running with admin rights")
