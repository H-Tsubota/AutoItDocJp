Keyword Reference

For...To...Step...Next

条件式に基づいてループします。

For <variable> = <start> To <stop> [Step <stepval>]
    statements
    ...
Next

 

パラメータ

variable カウントに使う変数
start カウントに使う変数の初期値
stop カウントに使う変数の終了値
stepval [オプション] ループを1回まわすごとにいくつカウントを増やすかの値（分数も可能）。デフォルトは1

 

注意

MustDeclareVarsがオンになっている場合でも、変数はLOCALで自動生成されます。

For...Next文は入れ子構造にできます。Forループはvariableがstopの値に達すると終了します。stepVal 、stop が変数の場合, その値は一番最初にループが実行される際に読み取られます。

次の場合はForループ内の命令文は実行されません。
   start > stop かつ step > 0, または
   start < stop かつ step がマイナスの値

 

関連

ContinueLoop, ExitLoop

 

例


For $i = 5 to 1 Step -1
    MsgBox(0, "Count down!", $i)
Next
MsgBox(0,"", "Blast Off!")
