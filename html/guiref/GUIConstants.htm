GUI Constants Include Files

AutoItで利用可能なGUI定数のインクルードファイルのリストです。

ファイル 関連するコントロール
#include <AVIConstants.au3> AVI
#include <ButtonConstants.au3> ボタン、グループ、ラジオ、チェックボックス
#include <ComboConstants.au3> コンボ
#include <Constants.au3> トレイ、Run、標準IO
#include <DateTimeConstants.au3> 日付け、月カレンダー
#include <EditConstants.au3> エディット、インプット
#include <GUIConstantsEx.au3> GUI作成、イベント
#include <ListboxConstants.au3> リスト
#include <ListviewConstants.au3> リストビュー
#include <ProgressConstants.au3> プログレス
#include <SliderConstants.au3> スライダー
#include <StaticConstants.au3> ラベル、画像、アイコン
#include <TabConstants.au3> タブ
#include <TreeviewConstants.au3> ツリービュー
#include <UpdownConstants.au3> アップダウン
#include <WindowsConstants.au3> Gui拡張スタイル
